Instalasi Debian

Debian adalah Operating Sistem (OS) berbasis linux yang sering digunakan untuk keperluan server, baik itu server DNS, Web Server dan masih banyak lagi.

Install dan Konfigurasi Service

Service adalah layanan yang dapat digunakan pada Debian agar server bisa digunakan sesuai peruntukannya. Service yang akan kita gunakan kali ini adalah Web Server, SSH, FTP server, dan DNS server
